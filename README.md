# 📚 Library Management System (C)

A simple **Library Management System** built using the **C programming language**.
This project demonstrates fundamental C programming concepts such as **structures, file handling, and menu-driven programs**.

The system allows users to manage library books through a command-line interface.

---

## 🚀 Features

* ➕ Add a new book
* 🔍 Search for a book
* 📖 Display all books
* 📕 Issue a book
* 💾 Store book records using **file handling**

---

## 🛠 Technologies Used

* **C Programming Language**
* **GCC Compiler (MinGW)**
* **File Handling**
* **Structures**
* **Command Line Interface**

---

## 📂 Project Structure

```
library-management-system/
│
├── library.c      # Main source code
├── library.txt    # File used to store book records
└── README.md      # Project documentation
```

---

## ⚙️ How to Compile and Run

### Compile the program

```
gcc library.c -o library
```

### Run the program (PowerShell)

```
.\library
```
